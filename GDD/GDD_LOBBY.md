# GDD_LOBBY

## Overview
Lobby is the pre-match configuration screen.
Host creates the room and controls all settings.
Other players join and select their character.

## Host
Host is the player who creates the room.

Host responsibilities:
- Configure all match settings
- Start the match when ready

## Player Count
Minimum: 2
Maximum: 6

## Character Selection
Each player selects a visual variant of their pawn.
Character selection is cosmetic only.
No gameplay impact.

## Board Selection
Early Access: Casino only.
Future updates will add new boards.

## Match Settings
Configured by host before the match starts.

### Match Length
Options:
- 10 turns
- 15 turns (default)
- 20 turns
- 30 turns

### Starting Coins
Configurable by host.
Default value: OPEN QUESTION

### Economy Options
Additional economy settings planned.
Exact options: OPEN QUESTION

### Trophy Configuration
- Enable/Disable Trophy Categories
- Change Trophy Count per Category
- Future Custom Rules

## Turn Order
Default: Random at match start.

## Open Questions
- Default Starting Coins value
- Full list of Economy options
- Voting system for Trophy configuration (future consideration)
- Turn order based on previous match results (future consideration)

## Canon Decisions
- Host controls all match settings.
- Character selection is cosmetic only.
- Match length is configurable.
- Starting Coins are configurable.
- Turn order is random by default.
- Early Access ships with Casino board only.
- Each player starts with Basic Dice only.
