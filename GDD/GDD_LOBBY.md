# GDD_LOBBY

## Overview
The Lobby is where players configure match settings before the game starts.
It allows customization of Trophy rules and future match options.

## Trophy Configuration
Players can adjust Bonus Trophy settings:
- Enable/Disable Trophy categories
- Change Trophy count per category
- Future custom rules
